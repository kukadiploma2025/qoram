# 🚕 VIP Taxi — Almaty Region (Sayakhat - Koram - Shelek)

Премиальное веб-приложение для организации поездок между Алматы, Корамом и Шелеком. Создано для удобства пассажиров и водителей, обеспечивая быструю связь без посредников.

## 🌟 Особенности проекта
* **Двусторонний интерфейс:** Раздельные кабинеты для пассажиров и водителей.
* **Умный выбор маршрута:** Быстрые кнопки для популярных направлений (Қорам, Шелек, Алматы).
* **Премиальный дизайн:** Современный Glassmorphism интерфейс с золотыми акцентами.
* **Прямая связь:** Водители могут звонить пассажирам в один клик прямо из приложения.
* **Безопасность данных:** Интеграция с Supabase для надежного хранения заявок.
