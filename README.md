# qoram
Премиальное приложение для поездок Алматы — Қорам — Шелек. Автор: Тлепберген Қуаныш (г. Астана, родом из Қорама, ру — Албан).  Проект создан для удобства земляков: пассажиры бронируют места в один клик, а водители видят список заказов в реальном времени. Glassmorphism дизайн, выбор мест и прямая связь. Сделано с заботой о родном крае!
